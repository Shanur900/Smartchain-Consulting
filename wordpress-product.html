Search

Search & Filter is a simple WordPress search and filtering plugin that improves the WordPress search box.

Remove the search box and use it as a filtering system for your posts and pages. You can search by Category, Tag, Custom Taxonomy, Post Type, Post Date, or any combination of these to further tailor your results. Dropdowns, checkboxes, radio buttons, and multi selections can all be used to display fields.

To personalize WordPress search and execute different types of searches on-site content, use our Search plugins to create an endless number of search forms and configure each search form individually.

Using the shortcodes, you may place the custom search forms in the site header, footer, sidebar or widget area, navigation or menu, pages, articles, custom post kinds, or anywhere else on the site.
